# Strawhat Cult

A sleek, minimal fan website dedicated to the legendary Straw Hat from One Piece.

## Features

- **Premium Design**: Deep ocean blue theme with gold straw hat accents
- **One Piece Themed**: Custom Straw Hat Jolly Roger logo, lore-accurate content
- **Straw Hat Generator**: AI-powered PFP generator using kie.ai
- **Smooth Animations**: Subtle fade-in effects and floating logo
- **Responsive**: Works on all devices

## Getting Started

### Prerequisites

- Node.js 18+
- npm or yarn

### Installation

```bash
npm install
```

### Environment Setup

Create a `.env.local` file in the root directory:

```
KIE_API_KEY=your_kie_ai_api_key_here
```

Get your API key from [kie.ai](https://kie.ai)

### Development

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000)

### Production Build

```bash
npm run build
npm start
```

## Structure

- `/src/app` - Next.js app router pages and API routes
- `/src/components` - React components
  - `Navigation.tsx` - Fixed navigation bar
  - `HeroSection.tsx` - Main hero with logo and title
  - `StorySection.tsx` - Origin of the Straw Hat lore
  - `MeaningSection.tsx` - What the hat represents
  - `GeneratorSection.tsx` - PFP generator with kie.ai
  - `CommunitySection.tsx` - Community links and CA
  - `Footer.tsx` - Minimal footer
